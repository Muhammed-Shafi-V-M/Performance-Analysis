## 👋 About the Project

This is my first data analysis project built using Microsoft Excel. I wanted to understand how different types of content perform across categories and time periods, and present those insights in a simple, visual way.

I worked with raw data and turned it into an interactive dashboard using pivot tables, charts, and slicers. The dashboard allows users to explore the data dynamically and get quick insights.
